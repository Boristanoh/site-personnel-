Site de l'ACPB4

Le site de l'Association des Classes Préparatoires Biologiques (ACPB) est un site statique développé avec HTML, CSS et JavaScript. Il présente les activités et ressources de l'association de manière claire et professionnelle. JavaScript améliore l'expérience utilisateur en automatisant les fonctionnalités interactives comme les formulaires et les animations.

Valkyrie

L'application de gestion des résidences de l'INP-HB a été conçue en Python avec Tkinter et CustomTkinter pour l'interface utilisateur. Elle utilise une base de données SQLite (sqlite3) pour gérer les données efficacement. L'application gère les attributions de chambres, enregistre les informations des chambres et suit les historiques d'attributions. Grâce à son interface intuitive, elle facilite la gestion quotidienne des résidences de l'INP-HB.

Binômage

Le Programme de Binômage de l'ACPB a été réalisé en équipe, où j'ai occupé le rôle de chef de projet. L'interface utilisateur a été développée avec Tkinter. Ce programme avait pour objectif d'attribuer à chaque élève de 1ère année un binôme de 2ème année, et de sauvegarder les résultats dans un fichier PDF pour une utilisation ultérieure.

Mutuelle

Cette interface utilisateur simplifie la gestion d'une mutuelle. Elle utilise une base de données SQLite3 pour enregistrer les données. Avec cette interface, vous pouvez enregistrer les activités, collecter les cotisations, ajouter de nouveaux membres et suivre toutes les modifications apportées à la base de données via les différentes tables.
